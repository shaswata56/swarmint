# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
(v2.1). In short: be respectful, assume good faith, and keep discussion focused
on the work. Harassment or discrimination of any kind is not tolerated.

Report concerns by opening a confidential issue or contacting the maintainers.
Maintainers may remove, edit, or reject contributions and comments that violate
this code.
